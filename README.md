# streamlit-quiz-project
## 나만의 동네 탐험 스타일 테스트

## 프로젝트 소개

Python과 Streamlit을 활용하여 제작한 웹 애플리케이션입니다.

사용자가 간단한 퀴즈를 풀면  
자신의 장소 탐험 성향을 분석하여 결과를 알려주는 앱입니다.

---

## 구현 기능

### 1. 첫 화면 학번 / 이름 표시
앱 실행 시 제출자의 학번과 이름이 가장 먼저 보이도록 구성했습니다.

### 2. 로그인 기능
- 아이디 / 비밀번호 입력
- 로그인 성공 / 실패 처리
- 로그인 상태 구분
- 로그인 취소 기능

### 3. 캐싱 기능
Streamlit의 `@st.cache_data`를 사용하여

- 퀴즈 데이터 불러오기
- 결과 데이터 불러오기

### 4. 퀴즈 기능
- 문제를 한 문제씩 표시
- 진행률 표시
- 답변 선택
- 결과 계산
- 최종 유형 출력

### 5. 결과 저장 기능
퀴즈 결과를 CSV 파일로 저장할 수 있도록 구현했습니다.

## 실행 방법

pip install -r requirements.txt
